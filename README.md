# DataBaseSystem Project

## 🧩 프로젝트 개요
TPC-H 데이터셋을 이용하여 `PART`와 `PARTSUPP` 테이블 간의  
Block Nested Loops Join 연산을 C++로 구현합니다.

## 📂 폴더 구조
- `src/` : 소스코드 (join.cpp 등)
- `data/` : TPC-H에서 생성한 입력 데이터(.tbl)
- `report/` : 보고서 및 발표자료
- `.gitignore` : 깃에 올리지 않을 파일 설정

## 🧑‍💻 개발 환경
- Ubuntu 24.04 (WSL)
- C++ (g++)
- Visual Studio Code
